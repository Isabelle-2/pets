Alimentador Automático

Dúvidas sobre Alimentador Automático

O alimentador automático é adequado para todos os tipos de animais? Esse alimentador em específico é para animais de porte pequeno e destinado a ração de 6mm.

O alimentador permite programar horários de alimentação específicos? Sim, através do Software será possível o dono controlar o horário.

O alimentador tem controle remoto ou é acessível via aplicativo? O alimentador será controlado através de um Software remotamente no seu smartphone.

Ele tem uma opção de alimentação manual? Sim, caso o dono prefira, pode desativar o modo automático e deixá-lo no manual.

Qual tipo de manutenção é necessária? Exige uma manutenção frequente na limpeza, principalmente nas partes removíveis.

Ele é silencioso ou pode causar ruídos durante a alimentação? Pode causar alguns ruídos, mas nada que cause desconforto tanto para o pet quanto para o dono.
